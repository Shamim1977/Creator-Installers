### Install

```
Windows:
Use Installer Interface

Mac:
Use Installer Inteface

Ubuntu:

deb:
sudo dpkg -i Creator_1.0.0_amd64.deb

snap:
sudo apt update
sudo apt install snapd
sudo snap install Creator_1.0.0_amd64.snap

appimage:
Just double click
```

### Uninstall

```
Windows: 
Run appwiz.cpl
Select
Uninstall

Mac:
Go to Applications
Drag Creator and drop on Trash

Linux/Ubuntu:
sudo apt remove Creator
```
